## What these docs cover

These documents describe the **public** Aether-9 model:

- source language structure
- compiler and bytecode workflow
- `.a9b` artifacts
- `.a9s` sidecars
- CLI commands
- VM execution model
- sandbox and policy behavior
- examples and evaluation flow

These docs intentionally do **not** expose private signing material, undisclosed secrets, or sensitive internal implementation details that should not be published in a public repository.
