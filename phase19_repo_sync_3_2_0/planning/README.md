# Aether-9 v3.2.0 Planning Pack

This planning pack defines the scope, priorities, sequencing, and release boundaries for **Aether-9 v3.2.0**.

## Release theme

**Integration-Ready Platform**

The purpose of v3.2.0 is not broad language expansion. Its purpose is to make Aether-9 easier to embed, integrate, automate, and evaluate in machine-readable workflows.

## Why this release exists

After v3.1.1, Aether-9 already has:
- a stable VM path
- binary and JSON artifacts
- CLI export / inspect / disasm / vm
- runtime diagnostics
- examples
- developer docs
- validation package
- packaging and repository sync

What is still missing is a cleaner **integration story**. That is the central problem v3.2.0 addresses.
