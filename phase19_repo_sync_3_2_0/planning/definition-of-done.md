# Definition of Done

Aether-9 v3.2.0 is done when all of the following are true.

## Product criteria
- A documented integration-facing API exists
- common workflows can be called programmatically
- key CLI commands support machine-readable output where planned
- artifact metadata/output story is clearer than in v3.1.1
- integrator docs and examples are present

## Engineering criteria
- tests cover new integration paths
- validation runner covers at least one integration-oriented path
- no regression in existing core runtime flows
- release notes and changelog are complete
- wheel builds and installs successfully

## Release criteria
- repo is synced
- version is bumped
- release notes are prepared
- test summary is recorded
- final package is installable
