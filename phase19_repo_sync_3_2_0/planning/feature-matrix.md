# Feature Matrix

## In scope
- Python integration API
- structured inspect output
- structured verify output
- structured vm execution result output
- machine-readable CLI modes
- artifact metadata cleanup
- integrator documentation
- embedding examples
- automation-oriented validation improvements

## Deferred
- richer type system
- float support
- map/dict support
- syntax growth
- optimizer work
- JIT
- non-Python host embedding beyond current priorities
- advanced policy DSL

## Explicitly excluded from v3.2.0
- broad language redesign
- VM rewrite
- Rust runtime completion as a public dependency for this release
- SCP coupling into the public package structure
- security claims beyond current validated public surface
