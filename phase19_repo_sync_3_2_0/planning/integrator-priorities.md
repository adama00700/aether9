# Integrator Priorities

## What integrators need most
1. Stable entry points for export / inspect / verify / run
2. Structured results consumable by scripts and services
3. Clear artifact metadata and validation reporting
4. Predictable exit behavior
5. Examples showing Aether as a subsystem, not only a CLI tool

## Implication for v3.2.0
Anything improving these five points is high priority.
